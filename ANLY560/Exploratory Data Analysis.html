Part Two: Observation from the Lag plots

From the figure below, we can observe that when lag=1, the data of season=2 obviously deviates from the diagonal, while the data of season=11 is most consistent with the diagonal, which may be because the data in our dataset is accumulated and reset every year. And since January of each year is not included in the data, only 11 months of data are counted each year, so when lag=1, season=2 is the data of the previous year at the end of the year, so the correlation between it and the original data lowest.

And this feature is more obvious in the following figures Lag1~Lag16 and Lag11 (correlation), we can find that when Lag=11, all the data can be better concentrated on the diagonal line, which shows that our The dataset has strong seasonal characteristics, and every 11 data is a period.

Part Four: ACF and PACF

We can see that the ACF plot exhibits a clear periodicity, which means that our data is not stationary. Meanwhile, in the PACF plot, we can also see that the correlation values are very high when lag=11 and lag=12. This shows that our data contains significant periodicity, which also shows that our data are not stationary.

1. Use Three appropriate MA windows for smoothing.(a small number, a little bit lager number and a high number)






2. Compare your smoothing for different MA windows and comment.

We can see that since the seasonal period in our time series is 11, when we set ma=3, the new time series are very close to the original time series, and when we adjust ma=5, we can see the new time series has reduced volatility compared to the original time series. Finally, when we set ma=10, which is very close to our seasonal period 11, we can see that the new time series has basically fluctuated slightly around the annual average.

Therefore, we found that when the value of MA is increased, the new time series can better reflect the overall trend in the original time series, while ignoring the fluctuations in the smaller interval.
